═══════════════════════════════════════════════════════════════════
                    USER JOURNEY EXAMPLES
═══════════════════════════════════════════════════════════════════

SCENARIO 1: New User Completing First Assessment
─────────────────────────────────────────────────
1. Start → Home Page
2. Click "Start Risk Assessment"
3. Fill form → Submit
4. View Results Page
   ├─ Can return Home
   ├─ Can start another assessment
   └─ Can view history (shows 1 assessment)

SCENARIO 2: Returning User Reviewing History
─────────────────────────────────────────────────
1. Login → Home Page
2. Click "My History"
3. View History Page (all past assessments)
   ├─ Can go back to latest results (if available)
   ├─ Can start new assessment
   ├─ Can search/filter assessments
   └─ Can return home

SCENARIO 3: User Wants to Compare Assessments
─────────────────────────────────────────────────
1. Complete assessment → View Results
2. Click "View All History"
3. See all past assessments
4. Click "Latest Results" to return
5. Click "Take Another" to create new
6. Fill form with different data
7. View new Results
8. Click "View All History" to compare
