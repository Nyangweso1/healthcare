# Health Tip Card Images

Add your health tip card background images in this folder.

Expected file names used by `templates/health_tips.html`:

- `checkups.jpg`
- `income_coverage.jpg`
- `dental_health.jpg`
- `mental_health.jpg`
- `cancer_screening.jpg`
- `health_literacy.jpg`

You can change the file names in `templates/health_tips.html` if needed.
